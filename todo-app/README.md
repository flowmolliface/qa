# TaskFlow — Todo App

SPA для управления задачами. Данные хранятся в `localStorage`.

## Режим Bug Hunt

**http://localhost:5173/?hunt=1** — приложение с намеренными багами для ручного тестирования.
См. [bug-hunt](../bug-hunt/).

## Запуск

```bash
npm install
npm run dev
```

Открыть http://localhost:5173

## Функции

- Добавление, удаление, отметка выполненных
- Фильтры: All / Active / Completed
- Очистка выполненных
- Валидация пустого названия

E2E-тесты: [e2e-tests](../e2e-tests/)
