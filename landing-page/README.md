# Landing Page

Статическое портфолио для GitHub Pages.

## Локальный просмотр

Открыть `index.html` в браузере или:

```bash
npx serve .
```

## GitHub Pages

Settings → Pages → Source: `/landing-page` (или root с index в корне).

Перед публикацией замените в `index.html`:
- ссылку на GitHub
- email, Telegram, GitHub в секции «Контакты»
