# Примеры тестовой документации

Реальные форматы артефактов QA: баг-репорты, чек-листы, тест-кейсы.

## Содержимое

- [bug-reports/](./bug-reports/) — баг-репорты разных severity
- [checklists/](./checklists/) — smoke и regression чек-листы
- [test-cases/](./test-cases/) — тест-кейсы в табличном виде

## Контекст

Документы описывают гипотетический сервис **TaskFlow** — веб-приложение для управления задачами (см. [todo-app](../todo-app/)).
