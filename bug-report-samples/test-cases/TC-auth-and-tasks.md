# Тест-кейсы: Auth и Tasks

| ID | Модуль | Название | Предусловия | Шаги | Ожидаемый результат |
|----|--------|----------|-------------|------|---------------------|
| TC-01 | Tasks | Создание задачи | Открыта главная | 1. Ввести «Купить молоко» 2. Enter | Задача в списке, поле очищено |
| TC-02 | Tasks | Пустое название | Открыта главная | 1. Enter без текста | Задача не создана, hint виден |
| TC-03 | Tasks | Toggle complete | Есть активная задача | 1. Клик по checkbox | Зачёркнутый текст, фильтр Active скрывает |
| TC-04 | Tasks | Удаление | Есть задача | 1. Клик × | Задача исчезла из списка |
| TC-05 | API | GET tasks | API запущен | GET /api/tasks | 200, массив JSON |
| TC-06 | API | POST invalid | API запущен | POST `{}` | 400, сообщение об ошибке |
| TC-07 | API | DELETE task | Задача id=X существует | DELETE /api/tasks/X | 200/204, задачи нет в GET |
