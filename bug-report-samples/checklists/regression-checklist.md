# Regression Checklist — TaskFlow v1.x

Запускать перед каждым релизом после smoke.

## UI — Список задач

- [ ] Длинное название (200+ символов) — обрезка или перенос без поломки вёрстки
- [ ] Быстрое добавление 10 задач подряд — без дублей и зависаний
- [ ] Счётчик «N items left» корректен при фильтрах
- [ ] Кнопка «Clear completed» удаляет только выполненные

## UI — Валидация

- [ ] Только пробелы в названии — ошибка
- [ ] Дублирование названия — разрешено (by design)

## API

- [ ] POST без поля `title` → 400
- [ ] PATCH несуществующего id → 404
- [ ] Content-Type: application/json обязателен

## Кросс-браузер (выборочно)

- [ ] Chrome (последний)
- [ ] Firefox (последний)
- [ ] Mobile viewport 375px
