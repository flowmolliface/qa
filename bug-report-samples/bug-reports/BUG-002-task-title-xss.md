# BUG-002: XSS в названии задачи при отображении списка

| Поле | Значение |
|------|----------|
| **ID** | BUG-002 |
| **Severity** | Critical |
| **Priority** | P1 |
| **Компонент** | Tasks / List |
| **Окружение** | Firefox 125, macOS 14 |
| **Статус** | Fixed in v1.2.1 |

## Шаги воспроизведения

1. Создать задачу с названием: `<img src=x onerror=alert(1)>`
2. Открыть список задач

## Ожидаемый результат

Текст отображается как строка, скрипт не выполняется.

## Фактический результат

Выполняется alert при рендере списка (innerHTML без экранирования).

## Root cause

Использовался `dangerouslySetInnerHTML` / отсутствие sanitization на клиенте.

## Fix verification

Повторить шаги на v1.2.1 — alert не появляется, текст экранирован.
