# BUG-001: Форма логина принимает пустой пароль

| Поле | Значение |
|------|----------|
| **ID** | BUG-001 |
| **Severity** | Major |
| **Priority** | P2 |
| **Компонент** | Auth / Login |
| **Окружение** | Chrome 124, Windows 11, staging |
| **Статус** | Open |

## Шаги воспроизведения

1. Открыть `/login`
2. Ввести email: `user@example.com`
3. Оставить поле «Пароль» пустым
4. Нажать «Войти»

## Ожидаемый результат

Кнопка неактивна или показывается ошибка «Введите пароль».

## Фактический результат

Запрос уходит на сервер, возвращается 400 без понятного сообщения пользователю.

## Дополнительно

- Воспроизводится стабильно
- Скриншот: `screenshots/BUG-001.png`
- Логи Network: POST `/api/auth/login` → 400
